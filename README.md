# RAPID (Rotation-Aware Probabilistic Inference Dashboard)

RAPID is an apllet designed to probabilistically infer stellar parameters from the synthetic populations. RAPID allows users to:
 * place target stars on the HR diagram, 
 * include asymmetric uncertainties in the inputted effective temperature and luminosity, 
 * visualise the resulting probability density, 
 * and extract posterior distributions for mass, age, and other stellar parameters.
